# 🚀 打包上线前必读

## ⚠️ 重要：打包前必须切换环境配置！

### 📍 快速切换步骤

#### 1. 修改 API 配置（必须）

**文件：** `src/config/aws-config.ts`  
**修改位置：** 第 41 行

```typescript
// 将这行：
const IS_LOCAL_DEV = true; // ⚠️ 上线前改为 false

// 改为：
const IS_LOCAL_DEV = false; // ✅ 生产环境
```

#### 2. 关闭开发模式（建议）

**文件：** `src/navigation/AppNavigator.tsx`  
**修改位置：** 第 39 行

```typescript
// 将这行：
const DEV_MODE_FORCE_ONBOARDING = true; // ⚠️ 上线前改为 false

// 改为：
const DEV_MODE_FORCE_ONBOARDING = false; // ✅ 生产环境
```

### ✅ 验证步骤

1. **修改配置**：将 `IS_LOCAL_DEV` 改为 `false`
2. **确认配置**：检查 `DEV_MODE_FORCE_ONBOARDING` 已关闭
3. **测试连接**：启动应用，确认能正常连接生产 API
4. **开始打包**：`expo build:ios` 或 `expo build:android`

| `DEV_MODE_FORCE_ONBOARDING` | `true`                    | `false`        |
| API 地址                    | `http://192.168.x.x:8000` | AWS Lambda URL |
